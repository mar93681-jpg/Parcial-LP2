Introducción

El presente trabajo grupal tiene como propósito aplicar los principios de la Programación Orientada a Objetos (POO) en el desarrollo de una librería estadística en Python. A través del uso de clases, herencia y polimorfismo, se busca crear un conjunto de herramientas que permitan calcular medidas estadísticas básicas tanto para datos cuantitativos como cualitativos.

El proyecto se desarrolló de manera colaborativa mediante la plataforma GitHub, promoviendo la organización del código, el control de versiones y el trabajo en equipo. De esta manera, se integran los conceptos teóricos de la POO con la práctica del análisis estadístico, fortaleciendo las competencias en programación y análisis de datos.

Objetivos

Objetivo general

Implementar un programa en Python que aplique los conceptos de Programación Orientada a Objetos para realizar análisis estadísticos de datos cuantitativos y cualitativos.

Objetivos específicos

  • Diseñar una clase base que gestione la validación y manipulación de los datos ingresados.
  • Aplicar herencia para crear clases derivadas que calculen medidas de tendencia central y dispersión para datos cuantitativos.
  • Implementar polimorfismo para adaptar los métodos de análisis según el tipo de variable (cuantitativa o cualitativa).
  • Desarrollar un proyecto colaborativo en GitHub aplicando buenas prácticas de documentación y control de versiones.
