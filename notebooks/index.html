Presentación de la Librería “MiLibreríaDemo”

Proyecto realizado por Hily y equipo — Curso de Lenguaje de Programación 2

Descripción

Esta librería fue creada para demostrar el uso de módulos en Python. Permite realizar operaciones matemáticas básicas y mostrar mensajes personalizados de forma sencilla y eficiente.

Instalación

pip install MiLibreriaDemo

Ejemplo de uso

from MiLibreriaDemo import operaciones

resultado = operaciones.sumar(5, 3)
print("El resultado es:", resultado)
